افکت

این یک نمونه از افکت است که میتواند به ریبایی سایتت کمک کند

افکت Parallax Scrolling یه تکنیک جالب تو طراحی وب هست که باعث میشه تصاویر پس‌زمینه نسبت به محتوای جلوتر با سرعت متفاوتی حرکت کنن، این حس عمق و پویایی به سایت میده. چطور این افکت رو توی سایتت پیاده کنی؟ با CSS و HTML خیلی راحت می‌تونی این کار رو انجام بدی. ۱. HTML (ساختار صفحه) افکت پارالاکس

افکت پارالاکس

این یک نمونه از افکت پارالاکس است که می‌تواند به زیبایی سایتت کمک کند.

۲. CSS (استایل‌دهی و ایجاد افکت) body, html { margin: 0; padding: 0; height: 100%; font-family: sans-serif; text-align: center; color: white; } .parallax { background-image: url('https://source.unsplash.com/1600x900/?nature,mountain'); /* لینک تصویر */ height: 100vh; background-attachment: fixed; /* ایجاد افکت پارالاکس */ background-position: center; background-repeat: no-repeat; background-size: cover; } .content { background-color: #222; padding: 50px; } چطور کار می‌کنه؟ background-attachment: fixed; باعث میشه تصویر پس‌زمینه ثابت بمونه و فقط متن جلو حرکت کنه. background-size: cover; تصویر رو متناسب با صفحه تنظیم می‌کنه. height: 100vh; باعث میشه بخش پارالاکس کل ارتفاع صفحه رو بگیره. می‌خوای حرفه‌ای‌ترش کنی؟ می‌تونی از چندین بخش پارالاکس استفاده کنی که با اسکرول صفحه تغییر کنن. می‌تونی با JavaScript افکت‌های پیشرفته‌تر مثل محو شدن (fade) هم بهش اضافه کنی. این نمونه رو تست کن، اگر خواستی جذاب‌تر بشه، بگو تا برات ارتقاش بدم!
